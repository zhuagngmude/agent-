# agent蜂群前端原型

这是从 `design/index.html` 演进出来的工程化静态前端原型。

## 文件

- `index.html`: 应用结构和入口。
- `styles.css`: 视觉样式、布局、响应式、状态样式。
- `data.js`: 页面导航、标题和后续 mock 数据入口。
- `app.js`: 页面导航切换逻辑。

## 设计原则

- 桌面端优先，适配 1440px 和 1920px 宽屏。
- 不做聊天页，不做营销页，不做 AI 大屏。
- 左侧主导航、顶部项目状态栏、中间工作区、右侧上下文面板。
- 自动代码执行页面必须突出安全边界、Git checkpoint、查看差异和用户确认。
- 中文内容必须使用真实 HTML 文本，不依赖图片生成。

## 运行

直接用浏览器打开：

```text
frontend/index.html
```

也可以在项目目录启动任意静态服务器。
