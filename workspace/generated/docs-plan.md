# 文档交接切片

## 1. 用户文档

### 1.1 项目概述
本项目提供一个简洁、美观的UI界面，用于展示和管理基础数据。用户可通过浏览器访问，无需安装额外软件。

### 1.2 功能说明
- **首页仪表盘**：展示关键数据概览和统计信息
- **数据列表页**：查看、搜索和筛选数据记录
- **详情页面**：查看单条数据的完整信息
- **基础操作**：支持数据的增删改查（CRUD）

### 1.3 使用指南
1. 打开浏览器，访问项目地址（默认：`http://localhost:3000`）
2. 登录系统（默认账号：admin / admin123）
3. 使用左侧导航菜单切换功能模块
4. 在数据列表页使用搜索框和筛选器快速定位数据
5. 点击“新增”按钮添加新数据，点击“编辑”修改现有数据

### 1.4 常见问题
- **Q: 页面加载缓慢？** A: 请检查网络连接，或刷新页面重试。
- **Q: 数据无法保存？** A: 请检查输入格式是否正确，必填项是否已填写。
- **Q: 忘记密码？** A: 联系管理员重置密码。

## 2. AI维护文档

### 2.1 项目结构
```
project/
├── src/
│   ├── components/    # UI组件
│   ├── pages/         # 页面组件
│   ├── services/      # API服务
│   ├── store/         # 状态管理
│   └── utils/         # 工具函数
├── public/            # 静态资源
├── package.json       # 依赖配置
└── README.md          # 项目说明
```

### 2.2 技术栈
- **前端框架**：React 18 + TypeScript
- **UI组件库**：Ant Design 5.x
- **状态管理**：Redux Toolkit
- **路由管理**：React Router v6
- **构建工具**：Vite 4.x
- **HTTP客户端**：Axios

### 2.3 关键模块说明

#### 2.3.1 路由配置 (`src/router.tsx`)
```typescript
const routes = [
  { path: '/', element: <Dashboard /> },
  { path: '/list', element: <DataList /> },
  { path: '/detail/:id', element: <DataDetail /> },
];
```

#### 2.3.2 API服务 (`src/services/api.ts`)
- `GET /api/data` - 获取数据列表
- `POST /api/data` - 新增数据
- `PUT /api/data/:id` - 更新数据
- `DELETE /api/data/:id` - 删除数据

#### 2.3.3 状态管理 (`src/store/dataSlice.ts`)
- `dataList` - 数据列表
- `currentData` - 当前选中数据
- `loading` - 加载状态
- `error` - 错误信息

### 2.4 维护注意事项
1. 修改组件时注意保持TypeScript类型定义同步
2. 新增API接口需在`services/api.ts`中添加对应方法
3. 状态管理变更需更新对应的Reducer和Action
4. 样式修改优先使用Ant Design的Token机制
