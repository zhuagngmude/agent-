# 文档交接切片

## 1. 用户文档

### 登录页面使用说明

#### 1.1 访问登录页面
- 打开浏览器，访问项目首页地址（如 `http://localhost:3000/login`）
- 页面将显示登录表单

#### 1.2 登录操作
1. 在“用户名”输入框中输入您的用户名
2. 在“密码”输入框中输入您的密码
3. 点击“登录”按钮
4. 系统验证通过后，将自动跳转到主页面

#### 1.3 常见问题
- **忘记密码**：请联系管理员重置密码
- **登录失败**：请检查用户名和密码是否正确，注意大小写
- **页面无法加载**：请检查网络连接或联系技术支持

## 2. AI 维护文档

### 2.1 登录页面组件结构
```
src/
├── pages/
│   └── LoginPage.jsx        # 登录页面主组件
├── components/
│   └── LoginForm.jsx        # 登录表单组件
├── services/
│   └── authService.js       # 认证服务（API 调用）
└── styles/
    └── login.css            # 登录页面样式
```

### 2.2 核心逻辑
- **表单验证**：前端对用户名和密码进行非空校验
- **API 调用**：通过 `authService.login(username, password)` 发送 POST 请求到 `/api/login`
- **状态管理**：登录成功后，将 token 存储到 localStorage，并更新全局认证状态
- **错误处理**：显示后端返回的错误信息（如“用户名或密码错误”）

### 2.3 配置项
- API 基础路径：`/api`
- Token 存储 key：`auth_token`
- 登录成功跳转路径：`/dashboard`
