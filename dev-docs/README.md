## 当前主线

```text
主控台输入目标
-> 总控选择项目 Agent 和专家
-> Runner preflight 边界检查
-> 自动生成/推进任务
-> 任务页查看总任务和 Agent 上下文
-> 可继续做 / 打开文件夹 / 删除任务
-> 输出目录 workspace/generated
```

当前主界面保留这些高频入口：

- 主控台
- 任务拆解
- 流程蓝图
- 运行输出
- AI 员工
- 审批与安全
- 系统设置

系统设置负责模型服务配置；密钥只进入当前桌面进程环境变量，不写入文档、数据库或日志。
