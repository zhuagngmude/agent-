## 当前主线

```text
主控台输入目标
-> 点击“全自动执行”
-> 自动生成/推进任务
-> 任务页查看总任务
-> 可继续做 / 打开文件夹 / 删除任务
-> 输出目录 workspace/generated
```

当前主界面只保留高频入口：

- 主控台
- 任务拆解
- 系统设置

系统设置负责模型服务配置；密钥只进入当前桌面进程环境变量，不写入文档、数据库或日志。
