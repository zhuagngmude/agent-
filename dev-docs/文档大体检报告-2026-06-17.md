## 当前真相

```text
主控台输入目标
-> 点击“全自动执行”
-> 自动生成/推进任务
-> 任务页查看总任务
-> 可继续做 / 打开文件夹 / 删除任务
-> 输出目录 workspace/generated
```

当前主入口：

- 主控台
- 任务拆解
- 系统设置

模型配置：

- 系统设置 -> 模型服务 / 智能体大脑
- API Key / Base URL / 模型 ID 写入当前桌面进程环境变量
- 输入框保存后清空
- 不落库、不写文档、不写日志

Runner 边界：

- 允许应用受控服务层内全自动执行。
- 禁止自由 shell、Git push/commit、文件删除、保护路径写入、提交密钥。
