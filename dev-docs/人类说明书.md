# agent蜂群 人类说明书

## 这是什么

`agent蜂群` 是一个多 AI 智能体协作控制台。当前版本先做本地 Web App + Mock / SQLite 试用，不开放真实 Runner、真实模型调用、云同步或完整权限系统。

## 现在能做什么

- 在工作流页输入项目想法和约束，生成本地确定性 `project_plan` 审批草案。
- 审批通过后，自动拆成 frontend / backend / qa / docs / reviewer 五个任务。
- 同时生成五条只读 Runner request queue 记录。
- 查看任务、审批、Runner 状态、Agent 配置审查和本地试用状态。

## 运行方式

SQLite 本地试用：

```powershell
powershell -ExecutionPolicy Bypass -File scripts/start-local.ps1
```

状态：

```powershell
powershell -ExecutionPolicy Bypass -File scripts/status-local.ps1
```

停止：

```powershell
powershell -ExecutionPolicy Bypass -File scripts/stop-local.ps1
```

Mock 开发：

```powershell
powershell -ExecutionPolicy Bypass -File scripts/start-dev.ps1
```
