## 前提检查

- [ ] Rust 工具链已安装（`rustc --version`、`cargo --version`）
- [ ] Tauri CLI 已安装（`cargo install tauri-cli` 或 `npm install -g @tauri-apps/cli`）

- [ ] `tauri.conf.json` 中前端入口指向 `packages/ui/dist/` 或 dev server `http://127.0.0.1:5173`
- [ ] 执行 `cargo tauri dev`，窗口正常打开，加载 packages/ui 页面
- [ ] 提交：`chore: init tauri project skeleton`
